MedStat Pro is a complete, self-contained, no-code web application designed for biomedical data science. It automates data cleaning, statistical testing, and report generation.

Features

Universal Import: CSV, Excel, SPSS (.sav), Stata (.dta).

Smart Statistics: Automatically selects between T-test, ANOVA, Chi-Square, etc.

Publication Ready: Generates "Table 1", Forest Plots, and Survival Curves.

Reporting: One-click HTML download with embedded high-res figures.

Installation & Running

Install Python 3.9+

Install Dependencies:
Open your terminal/command prompt and run:

pip install -r requirements.txt


Run the Application:

streamlit run medstat_pro.py


Access:
The app will open automatically in your browser at http://localhost:8501.

Privacy Note

This application runs locally or on your private server. No data is stored permanently; all data is cleared when the session ends.
